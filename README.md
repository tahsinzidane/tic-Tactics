# Tic-Tac-Toe

A simple Tic-Tac-Toe game built with Electron.js.

## Features
- Classic Tic-Tac-Toe gameplay.
- Built with Electron for desktop.

## Setup

- clone this repo
```
https://github.com/tahsinzidane/tic-Tactics.git
```

- Install dependencies
```
npm install
or
yarn add
```
- Run the program
```
npx electron .
```
 - Or run dist
 ```
 npm run dist
 ```
